Key Insight
Top 10 companies own 17% of global sows. Top 40 own 26% (pig333.com, 2024).
The remaining 74% are mid-size farms — the 500~5,000 sow segment is the most underserved market.

These farms are growing. 500 → 2,000 → 5,000 sows.
Free entry → natural paid conversion as they scale.

04
Tech Architecture
FastAPI + PostgreSQL + TimescaleDB. Completely new design, independent of legacy.
4.1 Backend: FastAPI (Python)
Recommendation: FastAPI(Python). AI features require Python. Starting with another tech means adding Python later = double work.
Java migration: Python is fast to build, Java is built to scale. Build fast now. Migrate when farms exceed 2,000 or enterprise SLA 99.9% is needed.
4.2 DB Design — Completely New
Key Message
CORE is completely independent from legacy systems.
Designing from scratch on global swine standards is faster and cleaner.

5.3 KPI Formulas
PSY = SUM(weaned piglets) [12mo] / AVG(active sows)
NPD = SUM(weaning-to-rebreeding interval) [12mo]
FCR = SUM(feed consumed kg) / SUM(weight gained kg)
Farrowing Rate = farrowed sows / mated sows × 100
