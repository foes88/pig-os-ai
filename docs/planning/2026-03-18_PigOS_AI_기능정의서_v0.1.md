## 모듈 A — 번식 사이클 관리

> **이게 양돈 SW의 존재 이유다.**
> 모돈 1두의 한 사이클: 교배 → 임신확인(28일) → 분만(114일) → 포유(21일) → 이유 → 휴지 → 재교배

### A-0. 번식 사이클 전체 흐름

```
[모돈 입식/등록]
      ↓
[발정 감지 / 교배 예정 알림]
      ↓
[A-1 교배 기록] ──→ AI 또는 자연교배
      ↓ (교배 후 21일)
[재발정 확인] ──→ 재발정이면 → 재교배로 이동
      ↓ (교배 후 28일)
[A-2 임신 확인] ──→ 음성이면 → 재교배 또는 도태 결정
      ↓ (교배 후 약 114일)
[A-3 분만 관리] ──→ 분만 기록 (산자수/사산/미라)
      ↓ (분만 당일)
[A-4 포유 관리] ──→ 위탁모 배정, 보조수유 기록
      ↓ (분만 후 21~28일)
[A-5 이유 기록] ──→ 이유두수, 이유체중, 이유일령
      ↓
[NPD 기간 시작] ──→ 발정 올 때까지 관리
      ↓ (이유 후 5~7일)
[재교배 예정 알림] ──→ A-1로 돌아감

[반복 사이클 — 산차 증가]
```

### A-1. 교배 관리

**화면 목적:** 교배 이벤트 기록. 교배 후 자동으로 임신확인 예정일(+28일), 분만 예정일(+114일) 생성.

**입력 항목:**
| 항목 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 모돈 번호 | 검색/스캔 | ✅ | 귀표 바코드 스캔 지원 |
| 교배 일자 | 날짜 | ✅ | 기본값: 오늘 |
| 교배 방법 | 선택 | ✅ | AI / 자연교배 / 복합 |
| 씨수퇘지 번호 | 검색 | AI시 필수 | 정액 코드 또는 씨수퇘지 ID |
| 교배 담당자 | 선택 | - | 로그인 사용자 자동 입력 |
| 교배 차수 | 선택 | - | 1차/2차/3차 |
| 비고 | 텍스트 | - | |

**자동 생성 (입력 후):**
- 임신확인 예정일 = 교배일 + 21일 (재발정 확인)
- 임신확인 예정일 = 교배일 + 28일 (초음파)
- 분만 예정일 = 교배일 + 114일
- 알림 등록: 임신확인 D-3, 분만 D-7, D-3, D-1

**비즈니스 규칙:**
- 현재 임신 중인 모돈은 교배 불가 (경고 표시)
- 이유 후 14일 이내 교배 시 주의 표시
- 산차 8차 이상 모돈 교배 시 도태 권고 알림

---

### A-2. 임신 확인 관리

**화면 목적:** 교배 후 임신 여부 확인 결과 기록.

**입력 항목:**
| 항목 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 모돈 번호 | 검색 | ✅ | |
| 확인 일자 | 날짜 | ✅ | |
| 확인 방법 | 선택 | ✅ | 초음파 / 육안 / 호르몬 검사 |
| 결과 | 선택 | ✅ | 임신 확인 / 재발정 / 미확정 |
| 예상 분만일 수정 | 날짜 | - | 기본값: 교배일+114일 |
| 비고 | 텍스트 | - | |

**결과별 처리:**
- **임신 확인** → 모돈 상태: "임신 중" / 분만 예정일 알림 등록
- **재발정** → 교배 재입력 화면으로 이동 권유 / 비생산일수(NPD) 자동 계산
- **미확정** → 2주 후 재확인 알림 등록

---

### A-3. 분만 관리

**화면 목적:** 분만 발생 시 산자 현황 기록. 양돈 SW에서 가장 중요한 입력 화면.

**입력 항목:**
| 항목 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 모돈 번호 | 검색/스캔 | ✅ | |
| 분만 일자 | 날짜+시간 | ✅ | |
| 총 산자수 | 숫자 | ✅ | 생존+사산+미라 자동 합산 |
| 생존 산자수 | 숫자 | ✅ | 카운터 UI |
| 사산수 | 숫자 | ✅ | 카운터 UI |
| 미라수 | 숫자 | ✅ | 카운터 UI |
| 평균 출생 체중 | 소수 (kg) | - | |
| 분만 형태 | 선택 | - | 자연분만 / 유도분만 / 제왕절개 |
| 분만 보조 여부 | Y/N | - | |
| 처치 내용 | 텍스트 | - | 옥시토신 투여 등 |
| 비고 | 텍스트 | - | |

**자동 처리:**
- 분만 기록 시 모돈 산차 +1
- 포유 시작일 = 분만일
- 이유 예정일 = 분만일 + 21일 (기본값, 수정 가능)
- 알림 등록: 이유 D-3, D-1

**비즈니스 규칙:**
- 분만 예정일 ±7일 이내가 아닌 분만 기록 시 경고
- 총 산자수 20두 초과 시 위탁모 배정 권유 알림
- 사산율 20% 초과 시 수의사 상담 권고 알림

---

### A-4. 포유 관리

**화면 목적:** 포유 기간 중 자돈 현황 관리. 위탁, 폐사, 보조수유 기록.

**입력 항목:**
| 항목 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 모돈 번호 | 검색 | ✅ | |
| 기록 일자 | 날짜 | ✅ | |
| 이벤트 종류 | 선택 | ✅ | 위탁 보냄 / 위탁 받음 / 폐사 / 압사 / 허약돈 |
| 두수 변동 | 숫자 | ✅ | |
| 현재 포유 두수 | 자동 계산 | - | 분만두수 - 누적폐사 + 위탁받음 - 위탁보냄 |
| 원인 | 선택 | - | |
| 비고 | 텍스트 | - | |

**일일 현황 조회:**
- 돈방별 현재 포유 두수
- 포유일령 (분만일 기준 자동 계산)
- 이유까지 남은 일수

---

### A-5. 이유 관리

**화면 목적:** 포유 종료(이유) 기록. 이후 모돈 재교배 사이클 시작.

**입력 항목:**
| 항목 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 모돈 번호 | 검색/스캔 | ✅ | |
| 이유 일자 | 날짜 | ✅ | |
| 이유 두수 | 숫자 | ✅ | |
| 이유일령 | 자동 계산 | - | 이유일 - 분만일 |
| 평균 이유 체중 | 소수 (kg) | - | |
| 이유 전 폐사 두수 | 숫자 | - | |
| 이유 방법 | 선택 | - | 일괄 이유 / 부분 이유 |
| 이유 후 모돈 처리 | 선택 | ✅ | 재교배 준비 / 도태 / 관찰 |
| 비고 | 텍스트 | - | |

**자동 처리:**
- 이유 기록 시 모돈 상태 → "이유 후 (NPD 시작)"
- 재교배 예정 알림 생성 (이유일 + 5일)
- PSY, NPD 자동 재계산

---

### A-6. 번식 성적 자동 계산

교배/분만/이유 데이터 입력 시 아래 지표 자동 계산:

| 지표 | 계산 공식 |
|------|-----------|
| **PSY** | (연간 이유두수 합계) ÷ 평균 활성 모돈수 |
| **NPD** | 365 - (평균 임신일수 × 복수) - (평균 포유일수 × 복수) |
| **분만율** | 분만두수 ÷ 교배두수 × 100 |
| **복당 이유두수** | 이유두수 합계 ÷ 분만 복수 |
| **포유 중 폐사율** | 포유 중 폐사두수 ÷ 총 출생두수 × 100 |
| **이유일령** | 이유일 - 분만일 (일수) |
| **산차별 성적** | 산차별 복당 이유두수, 분만율, PSY 비교 |

## 모듈 B — 모돈 개체 관리

### B-1. 모돈 목록

**필터/검색 조건:**
- 모돈 번호 (직접 입력 또는 바코드 스캔)
- 현재 상태: 임신중 / 포유중 / 이유후 / 공태 / 비육대기 / 도태
- 산차 범위
- 돈방 번호
- 품종
- 최근 교배일 범위

**목록 표시 항목:**
- 모돈 번호, 품종, 산차
- 현재 상태 (색상 배지)
- 마지막 이벤트 날짜
- 현재 상태 경과일수
- 다음 예정 이벤트 (교배예정/임신확인/분만예정/이유예정)
- 이상 여부 (NPD 초과, 재발정 등)

### B-2. 모돈 개체 상세

**표시 내용:**
- 기본 정보: 번호, 품종, 입식일, 입식 방법(자가 번식/구매/이입)
- 현재 상태 배지
- 전체 번식 이력 타임라인 (산차별 교배→분만→이유 그래프)
- 산차별 성적 요약 테이블
- 투약 이력
- 특이사항 메모

### B-3. 모돈 등록

| 품종 | 선택 | ✅ | LW, LD, Duroc, 기타 |
| 생년월일 (추정) | 날짜 | - | |
| 입식일 | 날짜 | ✅ | |
| 입식 구분 | 선택 | ✅ | 자가 번식(후보돈) / 외부 구매 / 이입 |
| 아비 정보 | 검색 | - | 자가 번식인 경우 |
| 어미 정보 | 검색 | - | 자가 번식인 경우 |
| 초도 산차 | 숫자 | - | 기본값: 0 (미경산) |
| 돈방 배정 | 선택 | ✅ | |
| 비고 | 텍스트 | - | |

### B-4. 도태·폐사 처리

**입력 항목:**
| 항목 | 타입 | 필수 | 비고 |

| 처리 일자 | 날짜 | ✅ | |

## 모듈 C — 비육돈 관리

> 번식돈(모돈)에서 생산된 자돈이 출하까지 관리되는 흐름

### C-1. 배치(Batch) 관리

### D-3. FCR 분석

**자동 계산 (배치별):**
- FCR = 사료 총 투입량 ÷ 총 증체량
- 목표 FCR 대비 현황
- 사료비 원가 (사료 단가 × 투입량)

### D-4. 두당 원가 분석

**계산 항목:**
| 항목 | 계산 방법 |
|------|-----------|
| 두당 사료비 | 총 사료비 ÷ 사육 두수 |
| 두당 약품비 | 총 약품비 ÷ 사육 두수 |
| 두당 생산비 | 사료비 + 약품비 + 기타 |
| 두당 수익 | 출하 수익 ÷ 출하 두수 |
| 두당 이익 | 두당 수익 - 두당 생산비 |
