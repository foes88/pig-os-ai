## 1. 한국

### 핵심 KPI
- PSY: 평균 21.9두(2023, 한돈팜스) / 상위 1% 32.5두 / 하위 30% 20.8두
- MSY: 18.8두 (PSY 대비 -3.1두 = 이유 후 폐사율 문제)
- FCR: 전체 평균 3.26~3.3 / 우수 농가 2.6 / 하위 3.8
- 분만율: 우수 농가 80~85%
- AI 비율: 상업 농장 90% 이상
- NPD 목표: 40일 미만 (공식 집계 부족)

### 사료
- 사료비 비중: 생산비의 55~60%
- 원료: 옥수수-대두박(수입 90%+)
- FCR 0.1 개선 = 두당 약 20원/kg 절감
- 단계별: 갓돈·젖돈·자돈·육성·비육·모돈 6단계

### 질병·방역 (최우선 관리)
- **ASF**: 2019년 최초 → 야생 멧돼지 지속 확산(2024년 누적 3,555건) / 2,300km 울타리 설치
- **PRRS**: 호흡기·생식기 질환의 56.7% / PRRSV-1·2 공존
- **FMD**: 2011년~전두 백신 접종
- 항생제: 성장촉진제 2011년 금지 / 치료용 수의사 처방 의무
- 2026~2030 국가 AMR 대책: WOAH 기준 mg/Animal Biomass 지표 2029년 도입 예정

### 시설·환경
- 무창(밀폐형) 돈사 표준
- **임신돈 스톨 → 군사 전환: 2030년 1월 의무**
  - 2025년 조사: 52.7% 농가 기한 내 전환 불가
  - 전환 시 수용 두수 약 46% 감소 (1.4㎡ → 2.3~2.6㎡)
- 여름 35°C↑, 겨울 -15°C, 봄·가을 일교차 10~15°C → 호흡기 질병 핵심 인자

### 출하·도체
- 출하 생체중: 110~115kg / 도체중 88kg / 출하일령 195일
- 등급: 1+, 1, 2, 등외 (축산물품질평가원)
- 1+ vs 2등급 가격차: kg당 603원 = 두당 약 53,064원
- 1등급 이상 비율: 70.3%
- 돼지고기 자급률: 72.8~73.7% / 1인당 소비 30kg (국내 육류 1위)

### 규정·디지털화
- **축산물이력제(mtrace)**: 출생~도축~소매 전 과정 의무 추적
- 한돈팜스: 전체 농가 72%(모돈 84.3%) 참여
- 스마트축사: 2034년까지 2,000호 목표 (2024년 돼지 4개 모델 선정)

### PigPlanCORE DB 설계 포인트
```sql
-- 한국 특화 필드 추가 필요
ALTER TABLE farms ADD COLUMN mtrace_id VARCHAR(20);          -- 이력제 농장 코드
ALTER TABLE sows ADD COLUMN stall_to_group_converted BOOLEAN; -- 군사 전환 여부
ALTER TABLE farrowings ADD COLUMN grade_1plus_ratio DECIMAL;  -- 1+등급 비율
ALTER TABLE health_events ADD COLUMN asf_zone_level INT;      -- ASF 위험 구역(1~3)
-- 도체 등급 테이블
CREATE TABLE slaughter_grades (
    id UUID PRIMARY KEY,
    sow_id UUID, pig_id UUID,
    grade VARCHAR(5),  -- 1+/1/2/등외
    weight_kg DECIMAL,
    slaughter_date DATE,
    price_per_kg DECIMAL
);
```

## 2. EU (덴마크·네덜란드·독일·스페인)

### 핵심 KPI
- PSY: 덴마크 **35.6두**(국가 평균, 2024) / DanBred 농장 36.2두 / 복당 이유 15.9두
- MSY: 덴마크 31~33두 / 네덜란드 28~30두 / 독일 26~28두
- FCR: 덴마크 **2.38**(2024, 상위10% 2.23) — 세계 최고
- AI 비율: 사실상 100%
- 모돈 갱신율: 덴마크 ~50%, EU 일반 40~50%

### 사료
- 사료비 비중: ~60% (2022 곡물 급등 시 70%+)
- 원료: 북유럽 = 밀-보리 기반 / 남유럽(스페인) = 옥수수-대두박
- 저단백 사료 + 합성 아미노산 전환 (질소 배출 저감 목적)

### 항생제 규제 — 세계 최엄격
- **2006년**: 성장촉진용 항생제 전면 금지
- **2022년**: 약용 산화아연(ZnO) 금지, 예방적 집단 항생제 투여 금지
- **2011~2022년**: EU 전체 항생제 판매량 **53% 감소**, 콜리스틴 81% 감소
- 덴마크: EU 평균보다 57% 낮은 사용량 / 옐로카드 시스템으로 초과 사용 농가 제재
- ESVAC 보고: 종별 mg/PCU 단위 의무 보고(2023~)

### 동물복지 규정
- EU 지침 2008/120/EC: **임신돈 군사 사육 의무** (2013년~ / 교배 후 4주~분만 전 1주)
- 비육돈 최소 면적: 85~110kg 기준 0.65㎡/두
- 환경 풍부화 자재(짚·건초) 제공 의무
- 덴마크: 분만틀도 자발적 루스 하우징 전환 중
- 인증 체계:
  - 네덜란드 **Beter Leven** (1~3별)
  - 덴마크 **Bedre Dyrevelfærd** (1~3하트)
  - 독일 **Tierhaltungskennzeichnung** (2023 의무화)

### 질병·방역
- ASF: 2024년 13개 EU 회원국 영향(83% 감소, 333건) / 루마니아 66% / 덴마크·NL·스페인 청정
- PRRS: 덴마크 **SPF 시스템**으로 농장 단위 관리(자발적)
- 덴마크 번식 농장: SPF 참여 사실상 필수

### 출하·도체
- **SEUROP 등급 체계**: 정육률(LMP)로 S(≥60%)~P(<40%) 분류
- 스페인: EU 최대 생산국(2024년 527만 톤, EU의 24%)
- 2024년 EU 전체 도축: 약 2억 2,100만 두

### 환경 규제 (추가 주의)
- EU 질산염 지침: 170kgN/ha 제한 → 네덜란드 2024년 모돈 **14% 감소**
- EUDR(산림벌채 규제): 대두 원료 조달에 영향

### PigPlanCORE DB 설계 포인트
```sql
-- EU 특화 필드
ALTER TABLE farms ADD COLUMN spf_status VARCHAR(20);          -- RED/BLUE (덴마크 SPF)
ALTER TABLE farms ADD COLUMN welfare_cert_level INT;           -- 1/2/3 (Beter Leven 등)
ALTER TABLE farms ADD COLUMN country_report_format VARCHAR(5); -- HIT/TAM/QS 등

-- 항생제 사용량 자동 계산
CREATE TABLE medicine_usage (
    id UUID PRIMARY KEY,
    farm_id UUID,
    medicine_name VARCHAR(100),
    active_substance VARCHAR(100),
    dose_mg DECIMAL,
    animal_count INT,
    usage_date DATE,
    ddda_value DECIMAL,      -- Defined Daily Dose Animal 자동 계산
    mg_pcu DECIMAL           -- mg/PCU 자동 계산
);

-- SEUROP 도체 등급
ALTER TABLE slaughter_records ADD COLUMN lmp_percent DECIMAL;  -- 정육률(%)
ALTER TABLE slaughter_records ADD COLUMN seurop_grade CHAR(1); -- S/E/U/R/O/P
```

## 3. 미국

### 핵심 KPI
- PSY: 평균 25.1두(PigCHAMP 2022) / 상위10% 31.7두 / 하위10% 19.2두
- MSY: 25~26두 / 상위10% 30두↑
- FCR: 상위 농장 2.5~2.7 / 업계 평균 ~3.0
- 분만율: 평균 78.3% / 상위 90.2%
- **모돈 폐사율 14.5%** (2014 대비 80% 상승 — PRRSV 1-4-4C, 군사 전환, 노동력 부족)
- AI 비율: 대규모 농장 90%+

### 사료
- 원료: 옥수수-대두박-DDGS(건조주정박)
- 사료비 비중: 55~65%
- 2024년 사료비: ~$36/cwt (2022~23 고점 대비 크게 하락)
- **락토파민(Paylean)**: FDA 승인 / 160개국+ 금지 → 수출용은 별도 무락토파민 라인

### 질병·방역 — PRRS가 최대 경제 질병
- **PRRS 연간 손실**: $6억 6,400만 달러
- MSHMP: 미국 모돈 ~50%(300만 두) 모니터링
- 공기 여과 시스템: PRRS 발생 위험 ~60% 감소
- ASF: 미발생 / 발생 시 10년간 500억 달러↑ 예상 손실
- 항생제: FDA GFI #213(2017) → 성장촉진 금지, VFD(수의사 사료처방) 의무화

### 시설·환경 — Prop 12 핵심
- **캘리포니아 Proposition 12** (2024.1 전면 시행)
  - 번식 모돈 최소 **24sqft(2.23㎡)** 요구
  - 캘리포니아 = 미국 돼지고기 소비의 15% / 모돈 사육 0.1% 미만
  - Smithfield·Tyson·JBS 등 대형사 준수 완료

### 출하·도체
- 출하 생체중: 270~290파운드(122~132kg)
- 도체 가치 기반(carcass merit) 구매 지배적 (공식 USDA 등급 아님)
- 현물 거래: 단 **2%** / 수식 가격: 약 60%
- 상위 4개 도축업체(Smithfield·JBS·Tyson 등): 전체 도축의 67% 점유

### 규정·인증
- PQA Plus: 돼지고기 품질보증 인증
- CSIA(공통 양돈 산업 감사): 바이어 요구 기준
- VFD 기록: 2년 보관 의무

### PigPlanCORE DB 설계 포인트
```sql
-- 미국 특화 필드
ALTER TABLE farms ADD COLUMN prop12_compliant BOOLEAN;         -- CA Prop 12 준수 여부
ALTER TABLE farms ADD COLUMN sqft_per_sow DECIMAL;             -- 두당 면적(sqft)
ALTER TABLE sows ADD COLUMN ractopamine_free BOOLEAN;          -- 무락토파민 여부

-- Feed Mill 연동 (미국 필수)
CREATE TABLE feed_deliveries (
    id UUID PRIMARY KEY,
    farm_id UUID,
    feed_mill_id UUID,
    feed_type VARCHAR(50),
    quantity_lbs DECIMAL,    -- 미국은 파운드 단위
    delivery_date DATE,
    cost_per_cwt DECIMAL,    -- cwt(100파운드) 단위 가격
    ractopamine_included BOOLEAN
);

-- VFD 기록
CREATE TABLE vfd_records (
    id UUID PRIMARY KEY,
    farm_id UUID,
    vet_license_no VARCHAR(30),
    drug_name VARCHAR(100),
    issued_date DATE,
    expiry_date DATE,        -- VFD 2년 보관 의무
    animal_class VARCHAR(50)
);
```

## 4. 남미 (브라질·멕시코)

### 핵심 KPI — 브라질
- PSY: 상위 계열화 농장 29~32두 (EU 중위권 근접)
- MSY: 상위 농장 26~28두
- FCR 목표: 2.3~2.6 / 상위 2.4 미만
- AI 비율: 계열화 농장 95%+

### 최대 강점: 세계 최저 생산비
- 생체중 kg당 생산비: **US$1.13**(마투그로수, InterPIG 2022)
  - 미국 $1.42 / 덴마크 $1.49 / 독일 $1.83 대비 압도적
- 사료 원료 자급: 세계 최대 대두 생산국 + 옥수수 2위
- 사료비 비중: 73~80% (절대 원가가 낮아 경쟁력 유지)

### 질병·방역 — 이중 청정 지위
- **ASF: 1984년 이후 청정** (전 세계 주요 생산국 중 유일)
- **PRRS: 청정** (두당 $5~6 비용 회피 효과)
- **구제역**: 2024년 백신 미접종 청정 지위 달성
- 멕시코: PRRS 상재국 / 2023~24년 상당한 피해

### 출하·수출
- 출하 생체중: 100~130kg
- 2023년 브라질: 스페인 제치고 **대중국 최대 돼지고기 수출국**
- 수출량 122.9만 톤(+9.8%), 수출액 **28.2억 달러**

### 계열화 구조 (BRF·JBS/Seara·Aurora)
- 80년+ 수직 계열화 역사
- 계약 농가(integrados): 유전자·사료·기술·시장 일괄 제공
- Topigs Norsvin: 브라질 시장점유율 34% (1위)

### 규정
- GTA(동물이동허가증): 모든 돼지 운송 의무
- SIF(연방)·SIE(주)·SIM(시) 3단계 검사 체계
- Agriness: 브라질 양돈 농장 **90%**, 라틴아메리카 200만 두+ 관리 / Cargill 투자

### PigPlanCORE DB 설계 포인트
```sql
-- 남미 특화 필드
ALTER TABLE farms ADD COLUMN integrator_id UUID;               -- 계열화 그룹
ALTER TABLE farms ADD COLUMN asf_free_certified BOOLEAN;       -- ASF 청정 인증
ALTER TABLE farms ADD COLUMN prrs_free_certified BOOLEAN;      -- PRRS 청정 인증

-- GTA(이동허가증) 연동
CREATE TABLE animal_movement_certs (
    id UUID PRIMARY KEY,
    origin_farm_id UUID,
    dest_farm_id UUID,
    pig_count INT,
    gta_number VARCHAR(30),  -- 브라질 GTA 번호
    issue_date DATE,
    validity_date DATE,
    ractopamine_free BOOLEAN -- 수출용 무락토파민 증명
);
```

## 5. 동남아시아 (베트남·태국·필리핀·인도네시아)

### 핵심 KPI
- PSY 상업 농장: 22~26두 (CP Foods 등)
- PSY 소농: **10~16두** (상업 농장 대비 60% 수준)
- FCR 상업: 2.4~2.8 / 소농: 3.5~4.5+
- AI 보급률: 상업 농장 80~95% / 소농 20~40%

### ASF — 최대 위기
- 베트남 2025년 1~3월: 127만 두↑ 살처분 (2024 대비 13배)
- **유전자형 I/II 재조합 변이주** 발견 → 기존 유전자형 II 백신 효과 제한
- 베트남 세계 최초 ASF 백신 3종 허가: NAVET-ASFVAC, AVAC ASF LIVE, DACOVAC-ASF2
- 태국·필리핀도 지속 발생

### 산업 구조 변화 (ASF 이후)
- 베트남: 농가 1,000만 호 → 200만 호로 급감 / 모돈 400만 두 → 220만 두(2023.3)
- 태국: 소농 18~19만 호 → 10만 호로 감소
- 필리핀: 전체 농가의 70.4%가 소농(2024.3)

### 사료
- 사료비 비중: 65~75%
- 베트남: 원료 65% 수입 (옥수수 950~1,000만 톤/년, 브라질이 최대 공급)
- 지역 부산물: 쌀겨·카사바·팜핵박·코프라박 활용
- 외국계 사료 기업(CP·Cargill·De Heus·New Hope)이 베트남 생산의 60% 차지

### 시설·특수 환경
- 소농: 개방형·반개방형 / 대농: 밀폐형·자동화

- 인터넷 인프라 불안정 → **오프라인 입력 필수**

### 출하·유통
- 출하 체중: 80~120kg (서구 대비 낮음)
- **전통 재래시장(wet market) 절대 다수**
- 콜드체인·가공육 비중 소수

### 지역별 특수사항

## 6. 중국

### 핵심 KPI
- PSY: 산업 평균 ~24.0두(2024 WEPIG) / 무위안 등 선도기업 26~28두↑ 목표
- 모돈: ASF 이전 4,400만 두 → 최저 2,000만 두 → 2023년 4,900만 두 → 2024년 4,078만 두
- FCR 업계 평균: 2.63(2024) / 상위 2.4~2.6
- 도축: 연간 약 **7억 260만 두**(2024)

### 사료 — 탈대두박 추진
- 사료비 비중: 57.2%
- 돼지 사료 생산: **1억 4,200만 톤**(2024, 전체 배합의 47%)
- **탈대두박 3개년 계획**: 13%(2023) → 10%(2030) 목표
  - 무위안: 이미 5.7% / 웬스(Wens): 3% 달성
- 옥수수 비율: 48.9%(2020) → 41.6%(2024) 하락, 밀·보리·수수 대체

### 질병·방역
- **ASF 2018.8 최초 발생 → 상재화** / 2018~19년 돼지·모돈 40~50% 상실
- ASF 백신: 아직 없음 (유전자결실·서브유닛·벡터 백신 연구 중)
- **2019년**: 사료 내 항생제 첨가 전면 금지
- 메가팜 생물보안: 군사 수준 (샤워인/아웃, 방호복, HD 카메라, 직원 기숙사, 공기 여과)

### 시설 — 세계 유일 다층 양돈 건물
- 후베이성 어저우: **26층 양돈 빌딩** (연간 120만 두 출하 능력)
- 광시 양샹그룹: 9~12층 / 베이징 신호프그룹: 5층
- 연간 성장률: 30%+ 추정 / 토지 부족 + 생물보안 강화가 동인

### 출하·시장
- 출하 생체중: ~100kg
- 2024년 평균 생체 가격: 17.08위안/kg (+10.9% YoY), 두당 이익 214위안
- **다롄상품거래소(DCE)** 생돈 선물: 2021년~ 가격 안정 기여
- 전략 비축돈육(냉동육): 정부 매입·방출로 가격 관리

## PigPlanCORE 아키텍처 핵심 요구사항

1. **모듈형 구조**: 지역별 규제·질병·인증 모듈 플러그인 방식 탈부착
2. **다중 단위계**: kg ↔ lb ↔ 근, ℃ ↔ ℉, 통화 자동 전환
3. **다중 벤치마크**: 국가별·지역별·상위10%·계열화 그룹 단위 비교
4. **규제 준수 자동화**: 항생제 보고(mg/PCU, VFD), 이력 추적, 환경 규제 자동 생성
5. **오프라인·저대역 지원**: SEA·중국 농촌 오프라인 입력·동기화
6. **API 확장성**: 한돈팜스, Agriness, PigCHAMP, PigVision과 데이터 교환
7. **AI 분석**: 벤치마크 이탈 조기 감지, 질병 예측, 출하 최적화
