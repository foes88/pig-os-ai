## 2. 경쟁사별 핵심 강점·약점 요약

### PigKnows (Valstone 인수, 2025.6)
**강점**
- 북미 1위 — 23개국 이상 사용
- Feed Mill 직접 발주·연동 (북미 농장 핵심 기능)
- Multi-Site + 본사-위탁 권한 구조 완성도 최고
- 영어/스페인어 이중 지원 (북미 현장 노동자 대응)
- 실시간 KPI 대시보드 + 벤치마킹

**약점/한계**
- **오픈 API 없음** (SoftwareWorld 확인: "No, PigKnows does not offer an API")
- Valstone 인수 후 폐쇄형 수직통합 심화 예고
- EU 규정 대응 기능 불분명
- 가격 비공개

---

### PigCHAMP (독립, 40년 브랜드)
**강점**
- 40개국+ 사용, 가장 오랜 브랜드 신뢰도
- 400개 이상 분석 필터 — 가장 강력한 분석 엔진
- 48개 이벤트 기록, 완전 커스터마이즈
- 유전 회사·Feed 회사·ESF 연동 표준화
- Grow-Finish(비육) 별도 모듈 보유
- RFID/바코드 모바일 앱

**약점/한계**
- Multi-Site 통합 관리 약함 (단일 농장 중심 설계)
- 오프라인 모바일: "Intermittent Mode" 부분 지원 (2024 신기능) — 완전 오프라인은 아님
- **오픈 API 없음** (SoftwareWorld 확인)
- 가격 비공개
- 구식 UI (설치형 레거시 베이스)

---

### Cloudfarms (BASF 인수, 2020)
**강점**
- **유일한 완전 클라우드 네이티브** (2013년 설계부터 클라우드)
- **24개 언어 지원** — 글로벌 최다
- 40개국+ 사용, 유럽·북미·아시아 균형
- 완전 오프라인 앱 + 자동 동기화
- Multi-Site 통합 보고 (JBS 17.5만 모돈 사례)
- 번식+승수+원핵 농장 전 단계 지원
- 2FA, 데이터 이중 백업 등 보안 강함

**약점/한계**
- BASF 인수 후 독립적 로드맵 불투명
- Feed Mill 직접 연동 기능 불명확
- EU 규정 특화 기능 표면적으로만 확인됨
- 가격 $500/년 시작 (리뷰 사이트 기준, 미공식)
- Add-on 패키지 구조 복잡

---

### MetaFarms (독립, 미국 중심)
**강점**
- **미국·캐나다 돼지 40%, 호주 50% 데이터 보유** → 사실상 벤치마킹 표준
- SOW + SOW ENTERPRISE + MAP 플랫폼 통합
- ESF(Nedap·AP Schauer) 연동 강함
- 오프라인 모바일 + 커스터마이즈 가능
- SMS 인수 후 벤치마킹·컨설팅 강화
- SwineTech·Jyga와 "Modern Tech Stack" 파트너십

**약점/한계**
- 북미 외 시장 취약 (유럽·아시아 미진출)
- Genetics/Pedigree 기능 약함
- EU 규정 대응 없음
- 오픈 API 부분적 (AgView 연동 정도)
- 가격 구 자료 기준 ($0.05~0.25/돼지/년) — 현행 미확인

## 3. PigPlanCORE 차별화 포인트 (경쟁사 공백 기반)

| 공백 영역 | 경쟁사 현황 | PigPlanCORE 기회 |
|---|---|---|
| **오픈 API** | PigKnows·PigCHAMP 없음, MetaFarms 부분적 | 완전 오픈 API — IoT·ERP·도축장 어디든 연동 |
| **완전 무료 진입** | 전 경쟁사 유료 (무료 체험만) | Phase 1 전 기능 무료 → 시장 선점 |
| **동남아 특화** | Cloudfarms만 베트남 진출, 나머지 없음 | 오프라인 + 다국어 + ASF 대응 + 무료 |
| **EU 규정 통합** | 모든 경쟁사 부분적 대응 | 항생제·동물복지·이력 추적 통합 모듈 |
| **가격 투명성** | 전 경쟁사 가격 비공개 | 유료 전환 시 투명한 Per Sow 가격 공개 |
| **한국 27년 노하우** | 해당 없음 | 피그플랜 데이터·운영 경험 기반 글로벌화 |
