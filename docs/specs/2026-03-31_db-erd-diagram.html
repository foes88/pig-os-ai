sows (Core Entity)

Status: GILT / ACTIVE / GESTATING / LACTATING / WEANED / DRY / CULLED / DEAD
Parity: 0 (gilt) ~ 20+ (increments on farrowing only)
exit_date: NULL = active, DATE = removed
Triggers: Status auto-transitions on event INSERT

breeding_cycles (v2 New)

Groups matings/farrowings/weanings per parity cycle.
UNIQUE(sow_id, parity)
mating_count: tracks re-services (1=first, 2+=retry)
One active cycle per sow enforced by partial UNIQUE index.

reproductive_events (v2 New)

8 event types: RETURN_TO_ESTRUS, ABORTION, EMPTY, INFERTILE, HEAT_DETECTED, SKIP_HEAT, ANESTRUS, NOT_IN_PIG
Essential for NPD 6-component calculation.

piglet_events (v2 New)

Tracks daily piglet changes during lactation:
DEATH, FOSTER_IN, FOSTER_OUT, STILLBORN_REMOVAL
Enables: fn_verify_piglet_balance()
View: v_current_nursing_piglets

Breeding Lifecycle — State Diagram

stateDiagram-v2
    [*] --> GILT : Entry (parity=0)
    [*] --> ACTIVE : Entry (parity>0, purchased)

    GILT --> GESTATING : 1st Mating
    ACTIVE --> GESTATING : Mating
    GESTATING --> LACTATING : Farrowing (parity+1)
    GESTATING --> ACTIVE : Return / Abortion
    LACTATING --> WEANED : Complete Wean
    LACTATING --> WEANED : Nurse Sow Wean
    WEANED --> GESTATING : Re-Mating
    WEANED --> LACTATING : Nurse Sow (foster-in)
    WEANED --> DRY : Extended rest
    DRY --> GESTATING : Mating

    GILT --> CULLED : Cull before mating
    GILT --> DEAD : Death before mating
    ACTIVE --> CULLED : Cull
    ACTIVE --> DEAD : Death
    GESTATING --> CULLED : Cull during pregnancy
    GESTATING --> DEAD : Death during pregnancy
    LACTATING --> CULLED : Cull during lactation
    LACTATING --> DEAD : Death during lactation
    WEANED --> CULLED : Cull after weaning
    WEANED --> DEAD : Death after weaning
    DRY --> CULLED : Cull
    DRY --> DEAD : Death

    CULLED --> [*]
    DEAD --> [*]

    note right of GESTATING
        Duration: ~114 days
        Check: Preg check at D+25
    end note

    note right of LACTATING
        Duration: ~21 days (EU min 28)
        Track: piglet_events daily
    end note

    note right of WEANED
        Target: Re-mate within 7 days
        NPD starts counting here
    end note

Auto-Transitions (Triggers)

matings INSERT → sow.status = GESTATING
farrowings INSERT → sow.status = LACTATING, parity+1
weanings INSERT → sow.status = WEANED
removals INSERT → sow.status = CULLED/DEAD, exit_date set

Blocked Transitions

CULLED → ANY (terminal, no recovery)
DEAD → ANY (terminal)
LACTATING → GESTATING (must wean first)
GILT → LACTATING (must mate + farrow first)

NPD-01: Gilt Entry → 1st Mating

View: v_npd_gilt_entry
Average: ~160 days
Warning if >210 days (late maturation)

NPD-02: Gilt Entry → Removal

View: v_npd_gilt_removal
100% wasted investment
Tracks reason: leg, health, infertility

NPD-03: WEI (Wean-to-Estrus)

View: v_sow_npd
Target: 4-7 days
>14 days = anestrus flag

NPD-04: Wean → Removal

View: v_npd_wean_to_removal
Sow culled after weaning without re-mating

NPD-05: Service → Re-Service

View: v_npd_reservice
Regular return: ~21d
Double return: ~42d
Irregular: investigate

NPD-06: Service → Removal

View: v_npd_service_to_removal
Pregnancy failed, then culled
Most expensive NPD type

Core (M0 + M1) — 19 tables

Must always be deployed together.
M1 (Sow) is the hub — 5 modules depend on sows table FK.
Cannot be split into microservices without API indirection.

Extensions (M2-M5, M8) — 16 tables

All reference sows or animal_groups via FK.
Can be feature-flagged on/off per farm.
Phase 1: M4 (Health) + M8 (KPI) only.

Independent (M6, M7, M9, M10) — 10 tables

Only reference farms table.
Can be deployed/scaled independently.
M9 (Market) has zero FK dependencies.
